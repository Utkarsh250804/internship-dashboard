3️⃣ Primary Key important kyu hai

Primary key:

Har record ko uniquely identify karti hai

Duplicate records prevent karti hai

NULL values allow nahi karti

Database me data integrity maintain karti hai

Example:

BookID = 1 → unique book
BookID = 2 → different book
Agar SQL me likhna ho to
CREATE TABLE Books (
BookID INT PRIMARY KEY,
Title VARCHAR(100),
Author VARCHAR(100),
Year INT,
Genre VARCHAR(50)
);
